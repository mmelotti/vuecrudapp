# A Simple Web Application

- Main Stack:

Vue, Express, Node, Mongo with Docker-compose setup;

- To Run (after clone):

Install Dependencies:

> npm install

Run the stack

> npm run dev

- To Run (with Docker):

Build Docker Images:

> docker-compose build

Run the stack:

> docker-compose up
